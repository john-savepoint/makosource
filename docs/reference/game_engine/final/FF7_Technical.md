# FF7/Technical

When trying to play Final Fantasy VII on PC, you can encounter a variety of problems. This guide describes the most common problems with their solutions.

## Installation Issues

Problems that occur before reaching actual gameplay:

### Error 112 During Installation

See [Error 112 during installation](FF7_Technical_Error_112.md) for resolution steps.

### AUTOEXEC.NT Error

If you get an error about `C:\WINDOWS\SYSTEM32\AUTOEXEC.NT` when launching the installer, see [AUTOEXEC.NT troubleshooting](FF7_Technical_Autoexec_nt.md).

### Hardware Rendering Unavailable

If you can't select Hardware rendering in game configuration, see [Hardware rendering configuration](FF7_Technical_Hardware_rendering.md).

### Game Won't Start

If you get an error while starting the game executable or the game won't run at all, see [Game startup troubleshooting](FF7_Technical_Game_wont_run.md).

### Black Screen After New Game

If after pressing 'New Game' you get only a black screen or the game crashes, see [New game black screen fix](FF7_Technical_New_game.md).

## Runtime Issues

Problems that occur during gameplay:

### Chocobo Race Crashes

If the game crashes when you try to enter Chocobo races, see [Chocobo race crash fix](FF7_Technical_Chocobo_races.md).

### Movies Played Upside Down

If movies are played upside down, see [Movie playback issues](FF7_Technical_Movies.md).

### Sound Blaster Audio Problems

If you have a Sound Blaster Live/Audigy card and experience sound problems, see [Sound Blaster audio fix](FF7_Technical_Sound_blaster.md).

### Poor Graphics Quality

If game backgrounds or text look low quality or corrupted, see [Graphics quality troubleshooting](FF7_Technical_Graphic_problems.md).

### First Fight Crash

If the game crashes after the first fight in a new game, see [First battle crash fix](FF7_Technical_1st_fight.md).

### Bike Minigame Too Fast

If the bike minigame speed is way too fast, see [Bike minigame speed fix](FF7_Technical_Bike_minigame.md).

### Random Crashes

If the game crashes in various locations, see [Random crash troubleshooting](FF7_Technical_Random_crashes.md).

### Battle Entry Crash

If the game crashes when entering a battle (during the swirl effect), see [Battle swirl crash fix](FF7_Technical_Swirl.md).

### No Movies Playing

If no movies are played at all, see [Movie playback disabled](FF7_Technical_NoMovies.md).
